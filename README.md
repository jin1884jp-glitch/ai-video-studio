# Agent workspace

This workspace contains project-level configuration and reusable skills for Codex-compatible agents.

## Layout

- `.agents/` — project agent metadata and local configuration.
- `.codex/` — Codex-specific project configuration.
- `agent-skill/` — reusable agent skills.
- `AGENTS.md` — instructions for agents working in this project.

The directories currently contain only placeholders. Add a skill or configuration when the project has a concrete need.
